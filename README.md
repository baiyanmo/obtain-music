# 酷狗音乐MP3下载器 - 使用说明

## 下载失败的常见原因

### 1. **VIP/付费歌曲** ❌
- **标识**: 搜索结果中没有 `[免费]` 标签
- **原因**: 需要酷狗VIP会员才能下载
- **解决**: 
  - 选择带 `[免费]` 标签的歌曲
  - 搜索同名的其他版本(可能有免费版)

### 2. **版权保护** ❌
- **原因**: 歌曲有严格的版权限制
- **提示**: "版权保护,无法下载"
- **解决**: 无法绕过,建议寻找其他平台

### 3. **区域限制** ❌
- **原因**: IP地址不在允许区域
- **提示**: "区域限制"
- **解决**: 无法绕过

### 4. **链接失效** ⚠️
- **原因**: API返回的下载链接已过期
- **提示**: "下载失败,状态码: 403/404"
- **解决**: 程序会自动尝试备用API

## 下载策略 (三层保障)

程序使用智能下载策略,自动尝试多个API:

1. **主API** (酷狗官方) → 速度快,音质好
2. **备用API** (移动端) → 成功率高,绕过部分限制  
3. **第三方API** (聚合) → 终极备选,可能绕过VIP限制

## 使用技巧

1. **优先选择免费歌曲**
   ```
   搜索结果中带 [免费] 标签的歌曲
   ```

2. **搜索多个版本**
   ```
   原版可能收费,但翻唱/纯音乐版可能免费
   示例: "月光下的凤尾竹 葫芦丝" → 多个版本可选
   ```

3. **批量下载技巧**
   ```
   输入: 1-10
   程序会自动跳过VIP歌曲,只下载免费的
   ```

4. **添加关键词**
   ```
   搜索: "歌名 纯音乐" 或 "歌名 伴奏"
   这类版本通常免费
   ```
